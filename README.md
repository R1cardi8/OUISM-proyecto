# Reportes Urbanos Ambientales - Santa Marta

Este proyecto es una aplicación web para registrar y consultar reportes relacionados con problemas urbanos y ambientales en la ciudad de Santa Marta. La idea es que cualquier persona pueda reportar situaciones como basura acumulada, contaminación, daños en espacios públicos, entre otros.

## 🚀 Tecnologías usadas

- Frontend: Vue.js
- Backend: Python (Flask / FastAPI según configuración)
- Base de datos: MySQL / PostgreSQL (según implementación)
- Comunicación: Axios

## 📌 Funcionalidades principales

- Crear reportes con descripción, ubicación e imágenes
- Listar reportes realizados
- Ver detalle de cada reporte
- Filtrado de reportes por categoría o estado
- Consumo de API REST desde el frontend
